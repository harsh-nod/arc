# Changelog

## 0.1.0

Initial public prototype release.

- ARC textual IR parser and printer
- Core verifier for SSA, types, structured regions, resources, proofs, and authority tokens
- Reference interpreter with execution traces
- Optimization passes with verifier checks
- Structured control-flow, async, invoke, and proof-erasure lowering
- Basic x86-64 ELF and WebAssembly code generation
- Capability declarations with effects, failures, and authority requirements
- Security audit, information-flow checks, taint checks, and sandbox policies
- Memory, proof, package, binary format, async, and dialect experiments
- GitHub Pages documentation and CI release checks
